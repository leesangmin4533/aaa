이 프로젝트는 BGF 리테일 시스템의 "중분류별 매출 구성비" 페이지에서 데이터를 자동 수집하기 위한 Python 기반 도구입니다.
Selenium을 통해 Nexacro 웹 애플리케이션에 로그인 후 JavaScript 스크립트를 주입하여 데이터셋을 직접 읽어 들입니다.
수집된 정보는 통합 SQLite 데이터베이스에 저장되며, (collected_at, product_code) 조합의 중복을 방지합니다.
`config.json` 파일을 통해 스크립트 경로와 DB 위치, 타임아웃 등이 관리되며, `main.py`는 과거 2일치 누락 데이터를 확인 후 오늘 데이터를 포함해 순차적으로 수집합니다.
로그인 모듈과 팝업 정리, DB 저장 유틸리티 등 각 기능이 모듈화되어 있으며, `tests` 디렉터리의 단위 테스트로 핵심 로직을 검증합니다.
